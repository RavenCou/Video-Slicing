# 输出要求
1. **序号**：从1开始连续编号，**必须输出所有镜头，不要限制数量**
2. **时段**：**每个镜头必须标注时间段**，格式为 MM:SS-MM:SS（分:秒-分:秒）
   - 根据关键帧的时间戳，推断每个镜头的起止时间
   - 第一个镜头从 00:00 开始
   - 每个镜头的结束时间就是下一个镜头的开始时间
   - 最后一个镜头的结束时间就是视频总时长
3. **台词**：**完整提取语音内容，不要删减或精简**，如果没有台词则标记为"[无台词]"
4. **景别**：从以下选项中选择最合适的 - 特写/近景/中景/全景/远景
   - 特写：局部细节（如手部动作、产品特写）
   - 近景：胸部以上，突出表情和动作
   - 中景：腰部以上，展现人物和周围环境关系
   - 全景：全身，展现完整人物动作
   - 远景：大范围场景，交代环境
5. **分镜名称**：用3-5个字概括该镜头核心内容（如"开场白"、"产品展示"、"操作演示"）
6. **画面描述**：30-100字，包含：
   - 主体（人物/物体）
   - 动作（正在做什么）
   - 环境/背景
   - 运镜方式（如有）

# 重要提示
- **【核心】ASR转录是完整的210秒口播音频，包含所有说话内容，这是台词的唯一准确来源，必须100%提取并合理分配到对应镜头中**
- **【核心】即使视觉画面是展示型（扫视空间、展示细节），如果ASR转录中有该时间段的口播内容，必须将这些台词分配到该镜头，不能标记为"[无台词]"**
- **必须基于所有关键帧和完整ASR转录结果，输出视频中的所有镜头**
- **每个镜头必须标注准确的时间段（MM:SS-MM:SS格式）**
- **台词必须完整保留，逐字逐句提取，不要精简或总结**
- **仔细观察相邻关键帧之间的细微变化，这些可能代表镜头切换**
- **场景、角度、构图、运镜的任何变化都应该是新镜头的开始**
- **展示型镜头（如扫视空间、展示细节）也要单独拆分为独立镜头**
- 镜头数量通常远超过10个，这是正常的，请完整输出所有镜头
